# AI-Enhanced Hybrid Beamforming for CSI Reduction in 5G

## Overview
This project implements an AI-based CSI compression framework using an Autoencoder to reduce feedback overhead in 5G systems.

## Technologies
- MATLAB (Channel Simulation)
- Python (Autoencoder Training)
- TensorFlow

## Project Structure
- `matlab/` → Channel generation
- `python/` → AI model
- `data/` → Dataset (not uploaded)

## Objectives
- Reduce CSI feedback overhead
- Maintain spectral efficiency
- Implement hybrid beamforming with AI
